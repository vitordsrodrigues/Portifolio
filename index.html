Meus Projetos

FinanFacil - Sistema de Planejamento Financeiro

FinanFácil é um sistema de planejamento financeiro pessoal que desenvolvi com foco na educação financeira e na mudança de hábitos de consumo. O usuário pode definir objetivos financeiros, registrar receitas, despesas e gastos com cartão, e acompanhar gráficos mensais por categoria. Além do controle financeiro, o site traz uma seção de conteúdos educativos para ajudar o usuário a entender melhor sua relação com o dinheiro e tomar decisões mais conscientes.

Tecnologias utilizadas: Node.js, Express, Sequelize, MySQL, HTML, CSS, Bootstrap e Handlebars, com arquitetura MVC e metodologia Kanban.

Site de Previsão do Tempo

Site de Previsão do Tempo que exibe as condições climáticas do dia em tempo real. O usuário pode buscar a previsão informando o CEP ou o nome da cidade, com dados fornecidos por uma API externa de clima.

Tecnologias utilizadas: HTML, CSS e JavaScript, com consumo de API para exibição dinâmica das informações.

Gerenciador de Tarefas

Gerenciador de Tarefas feito com Node.js, que permite cadastrar, editar, excluir e filtrar tarefas. Cada tarefa exibe o nome do usuário responsável, mas só pode ser editada ou excluída pelo próprio usuário. O sistema conta com login e registro, com controle de permissões por usuário.

Tecnologias utilizadas: Node.js, Express, Sequelize, MySQL, HTML, CSS e Handlebars.

Site de Literatura de Cordel

Cordel em Homenagem à Tecnologia é um site que apresenta o poema de Milton Duarte em formato de cordel digital, com foco na valorização da cultura popular e da linguagem poética. O site foi desenvolvido para dar destaque ao conteúdo de forma clara e responsiva.

Tecnologias utilizadas: HTML, CSS e JavaScript.
